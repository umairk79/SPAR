ONLINE LIBRARY

DESCRIPTION

Online library simulates a library by facilitating library functions

User will be able to perform functions like:

  1. Searching books
    • Search by genre
    • Search by Author
    • Search by rating
  2. Borrow/Return books
    • borrow book for certain period
    • return book within that period
    • pay library fee
